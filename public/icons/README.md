# PWA Icons

Para generar iconos placeholder (192x192 y 512x512), ejecuta:

```bash
npm run generate-pwa-icons
```

Para iconos finales, este directorio debe contener los iconos de la PWA en los siguientes tamaños:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png

Para generar los iconos, puedes usar herramientas como:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
- https://favicon.io/

Los iconos deben ser cuadrados y tener fondo transparente o sólido según tu diseño.
